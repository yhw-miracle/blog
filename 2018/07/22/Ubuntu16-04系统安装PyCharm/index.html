Ubuntu16.04系统安装PyCharm

随着人工智能的发展，python 越来越流行。作为年轻一代的互联网技术人员，学习 python 就变得十分必要。俗话说，“工欲善其事必先利其器”，在学习 python 之前，一个好的 IDE 将会使编程效率得到大幅度提高。本人在进行 python 开发时，强烈建议 pycharm 编辑器。

1. pycharm

pycharm 主要分为专业版和社区版，专业版是收费的，有 30 天试用期，当然专业版功能非常强大；而社区版是免费的，是专业版的阉割版，但基本上可以满足学习和开发的需要。pycharm 具有跨平台性，可以在 windows、linux、mac系统下使用。其官方链接为 PyCharm 。
